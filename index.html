About Ships - ROOT

It's a root POM: - it defines constants (versions mostly) - it has dependency and plug-in management configured - it has reporting configured

Project Modules

This project has declared the following modules:

Name Description
Ships - CLIENT It's Battleships game client. It is an isolated module having a light-weight client logic and GUI (JavaFX). It's a child POM: - it explicitly defines dependencies inherited from the parent POM - it independently defines its packaging
Ships - SERVER It's Battleships game server. It is an isolated module being mostly responsible for, - handling a network connection - handling message exchange - handling heavy-weight game rules It's a child POM: - it explicitly defines dependencies inherited from the parent POM - it independently defines its packaging
Ships - SHARED It's Battleships game infra API + Core. It is an isolated module having network utils and logging used both by server and client modules. It's a child POM: - it explicitly defines dependencies inherited from the parent POM
